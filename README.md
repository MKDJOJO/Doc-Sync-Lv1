# 强调内容

适合显示重要的提示信息，语法为

!> 内容。

```markdown
!> 一段重要的内容，可以和其他 **Markdown** 语法混用。
```

# 普通提示

普通的提示信息，比如写 TODO 或者参考内容等。

```markdown
?> _TODO_ 完善示例
```
